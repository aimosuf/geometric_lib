# Общее описание решения
Проект реализует математические функции для работы с геометрическими фигурами.  

- **Circle** — вычисление площади и периметра круга.
- **Square** — вычисление площади и периметра квадрата.

Функции могут использоваться:
- в учебных целях.
- как вспомогательные в других проектах.
- для бытрого решения математических задач.
- для веселья.

## Структура проекта
- `circle.py` — функции для работы с кругом  
- `square.py` - функции для работы с квадратом  
- `docs` — документация в формате Markdown
